## 设计理念
 - 使用清新，简洁的页面设计。给人一种既简单又不失内容，既清新又不缺风韵的视觉感受。我们从游——自然景致，食——美食诱惑，学——人文风情三方面，介绍我们心中的“最美大学”。

## 设计特点
 - 网页整体以武大樱园建筑群为背景，采用人见人爱的白色作为基础色调，简约朴素而不失优雅。每个模块均含丰富的图片与少量文字，给用户强烈的视觉冲击，以期达到“过目不忘”之效果。
